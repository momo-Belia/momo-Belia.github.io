选择排序

最基础的排序，核心思想是:把一串连续的元素先分未排序和已排序，然后在未排序的中找到最小(大)元素,把它放在序列的开头，现在它是已排的元素了。继续在未排的元素中找，直至有序。时间复杂度为\(O(n^2)\)

✈然后上例题,本题解用C++实现

  • 75. 颜色分类 题解:拿到题目先看数据，小于300，那么就可以用个\(O(n^{2})\)

  • 4. 寻找两个正序数组的中位数

    题解:我们上一题写过了select_sort，直接copy过来，然后再创一个函数，判断奇偶输出即可。

    • void func(vector& nums)是传入可变数组
    • insert可以拼接2个数组，手写则考虑二分
    • 利用位运算判断奇偶,n&1 = 1奇，反之偶

  • 747. 至少是其他数字两倍的最大数

    题解:最大元素并检查它是否 至少是数组中每个其他数字的两倍

    • 可能你会考虑用循环判断条件成立，但是排好序的从小到大的序列，倒数第二个就不行，那么就不用考虑了。
    • 排好序的序列序号会变，可以考虑哈希表，但很麻烦

  • 1502. 判断能否形成等差数列

    题解:记录下公差然后嘎嘎写一顿
